# Fake News Detection System

This project is a Machine Learning based Fake News Detection System that classifies news as Real or Fake using NLP techniques.

## Technologies Used
- Python
- Scikit-learn
- Pandas
- React.js
- TF-IDF Vectorizer

## Project Structure
- backend/ : ML model and API
- frontend/ : React frontend
- dataset/ : Kaggle dataset (not uploaded)

## How to Run

### Backend
```bash
pip install -r requirements.txt
python app.py
